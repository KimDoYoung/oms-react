- asis/application에 GWT/GXT 소스를 모두 가져다 놓았음.
- 주의할 것은 OMS시스템은 여러 회사들에게 서비스를 제공하는 SaaS프로그램임.
- 각 회사의 id를 확인하기 위해서 kova.localhost와 같이 호출하고 kova라는 회사 id를 추출하여 그 후 모든 db동작은 companyId를 바탕으로 움직이게 되어 있음.
- admin이라는 특별한 회사가 가상으로 존재하며 admin회사는 제작사(한국펀드서비스)를 의미함.
- 각 회사는 admin 권한을 가진 id가 존재하며 그 admin이 각 회사의 직원을 추가하고 직원별로 권한을 부여하는 체계임.
- admin이라는 회사의 admin이 각 개별회사의 admin 직원을 추가와 각 회사별 사용가능한 메뉴를 assing해 주는 방식임.
